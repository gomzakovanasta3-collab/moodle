= Режим редактирования

Для изменения содержимого объекта в Moodle, в том числе и курса, необходимо войти в *режим редактирования*. 

Сделать это можно 2-мя способами: 

. Перейти по ссылке «Режим редактирования» в блоке Настройки => Управление курсом.
. Нажать на кнопку «Режим редактирования» справа, под названием курса.

image::images/[Рис. 9. Команды для входа в режим редактирования]

После внесения изменений для выхода из режима редактирования нажмите на кнопку / ссылку «Завершить редактирование» в том же месте, где ранее находилась команда «Режим редактирования».
